What is juju?

A service orchestration tool for clouds and datacenters.

juju manages services, not machines

juju provides dynamic orchestration to connect services

juju incorporates pluggable provisioning

juju enables service definition reuse and management

Architecture basics

State servers and environment agents (api, provisioning, etc)

Machines with machine agents

Units agents running charm definitions

Charm Metadata

name: mediawiki
summary: "website engine for collaborative work"
description: A program for creating a collaboratively edited website.
requires:
  db:
    interface: mysql
  slave:
    interface: mysql
  cache:
    interface: memcache
provides:
  website:
    interface: http

options:
  title:
    default: ThinkUp
    type: string
    description: The name of the site
  timezone:
    default: UTC
    type: string
    description: The timezone for the server to use
  cache:
    default: false
    type: boolean
    description: Whether to cache pages or not
  cache-lifetime:
    default: 600
    type: int
    description: The time to cache a page for
